# iKnowBall
iKnowBall is a football dashboard web application that provides real-time data and insights across major football leagues. The app allows users to explore match fixtures, league standings, and top scorers through a clean and intuitive interface.

Features:
1] League Standings
View up-to-date rankings for top competitions like Premier League, La Liga, Serie A, Bundesliga, Ligue 1, and Champions League.
2] Match Tracking
Browse live and upcoming matches with filtering options.
3] Top Scorers
Check leading goal scorers across competitions.
4] Search Functionality
Quickly find teams and matches.
5] Filters & Sorting
Sort matches and filter by status for better insights.
